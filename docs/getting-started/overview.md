# Overview

Welcome to the Knowledge Base! This section will help you understand the structure and how to navigate through the information.

## Structure

The knowledge base is organized into three main sections:

1. **Documentation** - Core concepts and detailed explanations
2. **Guides** - Step-by-step instructions for specific tasks
3. **Resources** - Reference materials, tools, and templates

## Navigation Tips

- Use the **search bar** to quickly find specific topics
- The **sidebar** shows the current section's structure
- **Breadcrumbs** help you track your location
- **Previous/Next** links at the bottom help you navigate sequentially

## Getting Help

If you can't find what you need:
1. Try different search terms
2. Check the [FAQ](/docs/resources/faq)
3. Browse the [Glossary](/docs/resources/glossary) for terminology
4. Reach out to the knowledge base maintainers
