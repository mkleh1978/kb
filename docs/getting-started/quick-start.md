# Quick Start

Get up and running with the Knowledge Base in 5 minutes.

## Step 1: Explore the Structure

Familiarize yourself with the main sections:
- **Documentation** for concepts and reference
- **Guides** for how-to instructions  
- **Resources** for tools and templates

## Step 2: Use Search

The search bar at the top is the fastest way to find information. Try searching for:
- Specific topics or keywords
- Error messages or issues
- Tool names or features

## Step 3: Bookmark Important Pages

Keep frequently accessed pages handy:
- Use your browser's bookmarks
- Note important doc IDs for quick reference

## Step 4: Start Learning

Pick a topic that interests you and dive in. Recommended starting points:
- [Key Concepts](/docs/getting-started/concepts)
- [Common Tasks Guide](/docs/guides/common-tasks)

## Next Steps

Once you're comfortable navigating, you can:
- Explore advanced topics
- Contribute your own knowledge
- Suggest improvements
